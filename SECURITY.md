# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in PocketLantern, please report it responsibly.

**Do not open a public issue.**

Please use [GitHub's private vulnerability reporting](https://github.com/pocketlantern/pocketlantern/security/advisories/new) to submit your report.

We will respond within 24 hours and work with you to understand and address the issue.

## Scope

This policy covers the PocketLantern toolkit:

- MCP server (`apps/mcp-server/`)
- CLI (`apps/cli/`)
- Schema (`packages/schema/`)

## What is not in scope

- Content accuracy of knowledge cards (these are informational, not security-critical)
- Third-party dependencies (report upstream)
